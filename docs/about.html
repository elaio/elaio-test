El Proyecto

Este proyecto pertenece a una investigación realizada para el Trabajo de Fin de Máster con el título: "".

Se trata de un prototipo de un ejercicio de aprendizaje de idiomas, siendo L1 Español a L2 Inglés. Este ejercicio es enviado a una muestra de sujetos para que lo completen de manera anónima.

El programa registra eventos del usuario y los envía a un Learning Record System, en este caso, Learning Locker, utilizando la especificación de aprendizaje xAPI.

El objetivo de esta prueba consiste en:

  • Estudiar la recogida de datos relacionados con el aprendizaje de una lengua.
  • Investigar cómo recoger dichos datos utilizando la especificación xAPI.
  • Investigar cómo recoger dichos datos utilizando tecnologías web.

Creado por Elena Torró para el Máster en Lenguajes y Sistemas Informáticos de la UNED. 2017.
